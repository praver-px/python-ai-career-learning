## 核心内容

常见字段：

```text
timestamp 时间戳
nonce 随机数
sign 签名
token 身份或临时凭证
```

本地签名示例：

```python
import hashlib

raw = "timestamp=123&nonce=abc&secret=demo"
sign = hashlib.md5(raw.encode()).hexdigest()
```

真实工作流程：

```text
搜索参数名 -> 打断点 -> 看调用栈 -> 找输入字段 -> 本地复现 -> 验证
```
