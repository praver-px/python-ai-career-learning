# Week 17 - JS 加密参数与签名实验室

## 学习目标
- 理解 `sign`、`token`、`timestamp`、`nonce` 等参数的常见作用。
- 学会用浏览器 DevTools 搜索参数名、打断点、看调用栈。
- 能在本地测试环境中用 Python 或 Node.js 复现签名逻辑。

## 本周作业
- 创建 `projects/js_signature_lab/`。
- 自己写一个本地前端页面和后端接口，由前端 JS 生成签名，后端校验签名。
- 用 Python 或 Node.js 复现签名生成过程。
- 写一份分析笔记，说明参数来源、断点位置、调用链路。
- 不发布真实网站绕过细节。
