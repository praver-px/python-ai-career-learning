## 核心内容

建表：

```sql
CREATE TABLE article (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  title TEXT NOT NULL,
  url TEXT NOT NULL UNIQUE,
  published_at TEXT,
  content TEXT
);
```

Python 写入：

```python
import sqlite3

conn = sqlite3.connect("data.db")
conn.execute(
    "INSERT OR IGNORE INTO article(title, url) VALUES (?, ?)",
    (title, url),
)
conn.commit()
```

去重核心：给 URL 加唯一索引，重复写入时忽略或更新。
