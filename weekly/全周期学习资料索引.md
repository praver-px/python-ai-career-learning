# 全周期学习资料索引

这份文档解决“每周到底去哪里学”的问题。它不是任务清单，而是学习入口索引。

使用方式：

```text
1. 先打开对应周的 README。
2. 再打开本文件对应周。
3. 按“必看资料 -> 重点章节 -> 搜索关键词 -> 学完要会”的顺序学习。
4. 每周先看官方文档和项目内讲义，再让 AI 辅助解释。
```

注意：

```text
官方文档不要求逐字读完，只看本周列出的章节和主题。
遇到看不懂的地方，用 AI 解释，但最终必须自己写代码验证。
```
