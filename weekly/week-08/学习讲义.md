## 核心内容

Django 项目常见结构：

```text
manage.py
config/settings.py
config/urls.py
app/views.py
app/models.py
```

启动：

```bash
python manage.py runserver
```

最小 View：

```python
from django.http import JsonResponse

def health(request):
    return JsonResponse({"status": "ok"})
```
