## 核心内容

纯文本不适合直接入库，结构化 JSON 更适合业务系统：

```json
{
  "summary": "文章摘要",
  "keywords": ["Python", "Django"],
  "category": "backend"
}
```

用 Pydantic 校验：

```python
from pydantic import BaseModel

class ArticleAIResult(BaseModel):
    summary: str
    keywords: list[str]
    category: str
```
