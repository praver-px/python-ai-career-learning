## 核心内容

验证码不是第一选择。优先考虑：

```text
降低频率
缓存结果
减少重复请求
使用官方 API
确认授权和合规
```

本周只做本地样例实验，不对真实未授权网站做绕过。

ddddocr 基础：

```python
import ddddocr

ocr = ddddocr.DdddOcr()
with open("captcha.png", "rb") as f:
    result = ocr.classification(f.read())
print(result)
```
