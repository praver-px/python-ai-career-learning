## 核心内容

函数用于把一段逻辑命名。一个函数最好只做一件事：

```python
def normalize_title(title):
    return title.strip()
```

模块就是 `.py` 文件。把通用函数放到一个文件里，再在另一个文件导入：

```python
from pathlib import Path
import json
import csv
```

JSON 适合保存结构化对象，JSONL 适合一行一条数据，CSV 适合表格。

```python
article = {
    "title": "Python 入门",
    "url": "https://example.com/python",
    "published_at": "2026-07-02",
}
```
