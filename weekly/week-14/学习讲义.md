# Week 14 学习讲义：大模型 API 调用

本周目标：用 Python 调用 OpenAI 兼容 API，把 AI 能力接入程序。

## 学习资料

- OpenAI API：<https://platform.openai.com/docs>
- Text generation：<https://platform.openai.com/docs/guides/text-generation>
- API Reference：<https://platform.openai.com/docs/api-reference>
- python-dotenv：<https://pypi.org/project/python-dotenv/>

## 核心内容

关键概念：

```text
base_url
api_key
model
messages
stream
temperature
timeout
```

API Key 不写进代码，用 `.env`：

```text
OPENAI_API_KEY=xxx
OPENAI_BASE_URL=https://example.com/v1
```

## 每日安排

- Day 1：理解 API 调用结构。
- Day 2：配置环境变量。
- Day 3：完成一次非流式调用。
- Day 4：处理错误。
- Day 5：封装 `ask_ai()`。
- Day 6：写 `ai_explain.py`。
- Day 7：记录成本和失败样例。

## 练习题

1. 从环境变量读取 API Key。
2. 调用模型解释一个概念。
3. 让模型输出示例代码。
4. 捕获接口错误。
5. 记录请求耗时。
