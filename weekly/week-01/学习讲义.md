## 3. 第 1 天：环境、虚拟环境、运行 Python

### 3.1 Python 文件怎么运行

Python 脚本就是一个 `.py` 文件。

例如：

```python
print("hello python")
```

保存为：

```text
src/week01/hello.py
```

运行：

```bash
python src/week01/hello.py
```

### 3.2 为什么要虚拟环境

虚拟环境用于隔离项目依赖。不同项目可能需要不同版本的库。

创建：

```bash
python -m venv .venv
```

Windows PowerShell 激活：

```bash
.\.venv\Scripts\Activate.ps1
```

查看当前 Python：

```bash
python --version
where python
```

保存依赖：

```bash
pip freeze > requirements.txt
```

### 3.3 本周先不用安装第三方库

Week 01 尽量只用 Python 标准库：

```text
pathlib
collections
sys
```

这就是“避免重复造轮子”的第一课：文件路径用 `pathlib`，统计可以用 `dict` 或 `collections.Counter`。

## 5. 第 3 天：字符串

日志文件每一行都是字符串。

示例日志：

```text
127.0.0.1 - - [01/Jul/2026:10:00:00 +0800] "GET /api/articles HTTP/1.1" 200 1234
```

常用字符串操作：

```python
line = '127.0.0.1 - - "GET /api/articles HTTP/1.1" 200 1234'

print(line.strip())       # 去掉两边空白
print(line.split())       # 按空白切分
print("/api" in line)     # 判断是否包含
```

切分后：

```python
parts = line.split()
ip = parts[0]
status_code = parts[-2]
```

为什么 `parts[-2]` 是状态码？

因为这个样例最后两段通常是：

```text
200 1234
```

倒数第二个是状态码，倒数第一个是响应大小。

## 6. 第 4 天：list、dict、set、tuple

### 6.1 list：有顺序的一组数据

```python
urls = ["/", "/api/articles", "/login"]
urls.append("/api/users")
print(urls[0])
```

日志中，每一行可以放进 list：

```python
lines = ["line1", "line2", "line3"]
```

### 6.2 dict：键值对，最适合做统计

```python
ip_count = {}

ip = "127.0.0.1"
ip_count[ip] = ip_count.get(ip, 0) + 1

print(ip_count)
```

这句最重要：

```python
ip_count[ip] = ip_count.get(ip, 0) + 1
```

含义：

```text
如果 ip 已存在，取原来的次数加 1。
如果 ip 不存在，默认从 0 开始加 1。
```

### 6.3 set：去重

```python
unique_ips = set()
unique_ips.add("127.0.0.1")
unique_ips.add("127.0.0.1")
print(unique_ips)
```

### 6.4 tuple：不可变的一组数据

```python
record = ("127.0.0.1", "/api/articles", "200")
```

Week 01 重点是 `list` 和 `dict`。

## 8. 第 6 天：函数和文件读取

### 8.1 为什么要写函数

不要把所有代码写在一起。函数让代码更清楚。

错误写法：

```python
# 所有逻辑堆在一起，后面不好改
```

推荐拆分：

```python
def read_lines(file_path):
    ...

def parse_line(line):
    ...

def count_items(items):
    ...

def main():
    ...
```

### 8.2 pathlib 读取文件

```python
from pathlib import Path

path = Path("weekly/week-01/sample_access.log")
text = path.read_text(encoding="utf-8")
lines = text.splitlines()

print(len(lines))
```

为什么用 `pathlib`？

```text
它是 Python 标准库，专门处理路径，比字符串拼路径更稳。
```

### 8.3 解析单行日志

先用简单方式：

```python
def parse_line(line):
    parts = line.split()
    if len(parts) < 9:
        return None

    ip = parts[0]
    method = parts[5].strip('"')
    url = parts[6]
    status_code = parts[-2]

    return {
        "ip": ip,
        "method": method,
        "url": url,
        "status_code": status_code,
    }
```

注意：这只是学习版解析器，适合固定样例日志。真实生产日志要用更严格的解析方式。

## 9. 第 7 天：完成 log_analyzer.py

推荐结构：

```python
from pathlib import Path
import sys


def read_lines(file_path):
    path = Path(file_path)
    return path.read_text(encoding="utf-8").splitlines()


def parse_line(line):
    parts = line.split()
    if len(parts) < 9:
        return None

    return {
        "ip": parts[0],
        "method": parts[5].strip('"'),
        "url": parts[6],
        "status_code": parts[-2],
    }


def add_count(counter, key):
    counter[key] = counter.get(key, 0) + 1


def print_top(title, counter, limit=10):
    print(f"\n{title}")
    for key, count in sorted(counter.items(), key=lambda item: item[1], reverse=True)[:limit]:
        print(f"{key}: {count}")


def analyze(file_path):
    lines = read_lines(file_path)

    ip_count = {}
    status_count = {}
    url_count = {}

    for line in lines:
        record = parse_line(line)
        if record is None:
            continue

        add_count(ip_count, record["ip"])
        add_count(status_count, record["status_code"])
        add_count(url_count, record["url"])

    print(f"total lines: {len(lines)}")
    print_top("top ip", ip_count)
    print_top("status code", status_count)
    print_top("top url", url_count)


def main():
    if len(sys.argv) != 2:
        print("usage: python src/week01/log_analyzer.py weekly/week-01/sample_access.log")
        return

    analyze(sys.argv[1])


if __name__ == "__main__":
    main()
```

你可以照着写，但不要只复制。至少要自己回答：

```text

add_count 为什么用 dict.get？
print_top 里 sorted 是怎么排序的？
main 为什么检查参数数量？
```
