# 项目路线

## 项目 1：健壮版新闻采集器

目标岗位能力：

- Python 基础
- requests
- HTML 解析
- SQLite
- 日志
- 重试

功能：

- 配置起始 URL。
- 采集列表页。
- 解析标题、链接、时间。
- 保存到 SQLite。
- URL 去重。
- 失败 URL 记录。

简历写法：

```text
实现新闻采集器，支持请求超时、失败重试、断点续爬和 URL 去重，采集结果入库 SQLite，并通过日志记录采集状态。
```

目标岗位能力：

- Django
- ORM
- Middleware
- API
- 后台任务

功能：

- 文章管理。
- 采集任务管理。
- 任务状态。
- 统一响应。
- 请求日志中间件。

简历写法：

## 项目 3：AI 文章摘要与标签系统

目标岗位能力：

- LLM API
- Prompt
- 结构化输出
- 数据处理

功能：

- 读取文章。
- 调用 AI 生成摘要。
- 生成标签。
- JSON 结构化输出。
- 结果入库。

简历写法：

```text
接入 OpenAI 兼容大模型 API，为采集文章自动生成摘要和标签，使用结构化 JSON 输出并落库，提升内容检索和分类效率。
```

## 项目 3.5：验证码 OCR 实验室 captcha_lab

目标岗位能力：

- 验证码识别工具使用
- ddddocr
- OCR 准确率评估
- 内部服务封装
- 合规边界意识

功能：

- 读取本地验证码图片样例。
- 使用 ddddocr 进行识别。
- 统计准确率、耗时、失败样例。
- 封装 FastAPI 或 Django 内部测试接口。
- 明确说明只用于授权测试和本地样例。

简历写法：

```text
实现内部验证码识别实验服务，基于 ddddocr 对本地验证码样例进行识别和准确率评估，并封装为 HTTP API，支持失败样例归档和识别结果统计。
```

---

## 项目 3.6：JS 签名参数实验室 js_signature_lab

目标岗位能力：

- JS 基础
- DevTools 断点调试
- 加密/签名参数分析
- Python/Node.js 复现签名

功能：

- 本地前端生成 `sign`。
- 后端校验 `timestamp`、`nonce`、`sign`。
- 使用 DevTools 定位签名生成逻辑。
- 使用 Python 或 Node.js 复现签名。
- 输出完整分析笔记。

简历写法：

```text
搭建 JS 签名参数分析实验环境，模拟前端签名生成与后端校验流程，使用浏览器断点和调用栈定位签名逻辑，并用 Python/Node.js 复现签名生成过程。
```

## 项目 4：个人学习知识库 RAG

目标岗位能力：

- RAG
- 文档切分
- 检索
- AI 问答

- 读取 Markdown 学习文档。
- 按标题切分。
- 关键词或向量检索。
- AI 基于检索内容回答。
- 返回引用来源。

简历写法：

```text
实现基于 Markdown 文档的轻量 RAG 问答系统，支持文档切分、相关片段检索和基于上下文的问答生成。
```
