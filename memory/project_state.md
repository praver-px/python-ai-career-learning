## 项目定位

本仓库是一个面向 Python 后端、Django、爬虫、AI 应用开发和求职准备的长期学习仓库。

目标不是单纯“学 Python”，而是形成组合能力：

```text
Python 工程基础
  + 爬虫 / 数据采集 / 反爬分析
  + Django / DRF / 公司魔改框架阅读
  + 数据库 / Redis / 任务调度 / 部署
  + AI 应用开发 / RAG / Agent / Workflow
  + 简历项目 / 面试 / 投递
```
