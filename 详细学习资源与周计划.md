# 详细学习资源与周计划

这份文档用于解决“到底学什么、看哪里、看到什么程度、做什么作业”的问题。

学习顺序固定为：

```text
官方文档/权威资料 -> 查已有轮子 -> AI 辅助解释 -> 自己写最小练习或复用成熟方案 -> 测试验证 -> 复盘提交 Git
```

每周最低交付：

```text
notes/week-xx.md                 # 学习笔记
weekly/week-xx/README.md         # 本周复盘
src/weekxx/ 或 projects/对应项目  # 代码
至少 1 次 git commit
```

每周还必须记录：

```text
本周复用了哪些标准库、框架能力、公司封装或第三方库
哪些地方原本想自己写，后来发现已有方案
为了理解原理手写了哪个最小版本
为什么这个手写版本不直接用于生产
```
