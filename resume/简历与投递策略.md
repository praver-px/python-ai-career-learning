# 简历与投递策略

## 1. 简历定位

推荐标题：

```text
Python 后端开发 / 数据采集 / AI 应用开发
```

不要只写：

```text
爬虫工程师
```

因为岗位面会窄。

---

## 2. 技能栏模板

```text
编程语言：Python，熟悉函数、模块、异常处理、日志、pytest 基础。
Web 后端：熟悉 Django、Django ORM、Middleware、DRF 基础，能开发 REST API。
数据采集：熟悉 requests、Session、Cookie、Header、XPath、BeautifulSoup、Scrapy/Playwright 基础。
数据库：熟悉 MySQL/SQLite 基础，了解索引、去重、分页查询。
AI 应用：了解 OpenAI 兼容 API、Prompt、结构化输出、RAG 基础，能将 AI 能力接入 Django 项目。
工程工具：Git、Linux 基础、虚拟环境、requirements、日志排查。
```

---

## 3. 项目描述模板

项目名：AI 采集学习平台

```text
项目描述：
基于 Django 实现的内容采集与 AI 学习辅助平台，支持创建采集任务、抓取文章、入库管理、AI 摘要、标签生成和学习题生成。

技术栈：
Python、Django、SQLite/MySQL、requests、BeautifulSoup/XPath、OpenAI-compatible API、logging。

职责：
1. 设计 Article、SpiderTask 等数据模型，实现 URL 去重和任务状态管理。
2. 封装采集服务，支持 timeout、失败重试、异常日志记录。
3. 封装统一响应和请求日志中间件，提升接口可维护性。
4. 接入大模型 API，实现文章摘要、关键词提取和学习题生成。
5. 编写 README 和接口说明，沉淀部署和使用流程。
```

## 5. 面试准备重点

### Python

- list/dict/set 区别。
- 生成器。
- 装饰器。
- 异常处理。
- logging。
- 多线程/异步基础。

### 爬虫

- requests timeout。
- Session/Cookie。
- 403/429 怎么处理。
- XPath/CSS。
- 去重。
- 断点续爬。
- 反爬和合规。

### Django

- 请求生命周期。
- Middleware。
- ORM 查询。
- migration。
- DRF Serializer。
- 统一异常和响应。

### AI

- Prompt 怎么设计。
- 什么是 RAG。
- 如何控制模型输出 JSON。
- 如何避免幻觉。
- API key 如何安全管理。
